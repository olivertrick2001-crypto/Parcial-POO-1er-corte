Parcial Primer Corte: Sistema de Evaluación de Préstamos

Este trabajo consiste en un programa en C++ que automatiza la aprobación de créditos bancarios. 

El código evalúa si un préstamo es viable analizando:
* **Perfil del cliente:** (Estudiante, Empleado, Pensionado o Desempleado).
* **Restricciones:** Montos máximos permitidos y plazos límite.
* **Condiciones especiales:** 1. Aumento de tasa por plazos mayores a 36 meses.
    2. Descuento en intereses para clientes con codeudor (empleados/pensionados).

Es un ejercicio de lógica de programación aplicado a reglas de negocio financieras.
